Descobrindo Tags

<header> A tag <header> representa o cabeçalho da página, podendo conter logotipos, elementos navegacionais, formulários de pesquisa, entre outros.

<section> A tag <section> representa uma seção genérica da página, geralmente utilizada quando não há opções mais semânticas.

<var> A tag <var> é utilizada para representar variáveis em contextos de operações matemáticas ou de códigos de programação, exemplo: nome = "Gabriel Lara"

<footer> A tag <footer> representa o rodapé da página, podendo conter informações sobre, os direitos autorais, o autor e links para documentos relacionados.

<blockquote> Indica que o texto é uma citação longa.

<textarea> Área de preenchimento de texto, permite um texto ou uma mensagem no seu interior, pode ser redimensionado

<pre> Insere textos pré-formatados

<canvas> Geralmente usado para desenhar e animar utilizando javascript

<select> O elemento HTML select (<select>) apresenta um menu de opções. É um elemento auxiliado por outro 'option'(<option>) e cada um desses 'option' guarda uma opção para o usuário escolher na página. As opções podem também, ser pré-selecionadas para o usuário como 'escolha' assim como se vê nas paginas que vemos no dia-dia.

<figure> O elemento HTML <figure> representa o conteúdo independente, freqüentemente com o elemento (figcaption), o qual é usado exclusivamente para legendas. Ele é usado para peças que esta isolado corpo principal da página, é independente do fluxo principal.Normalmente, isso é uma imagem, uma ilustração, um diagrama, um trecho de código, mas que pode ser movido para outra página sem afetar o entendimento da página.

<output> O elemento de saída (<output>) é um elemento no qual um site ou aplicativo pode colocar os resultados de um cálculo, através do elemento input, que deve, ou devem, ser preenchidos para a resposta aparecer automaticamente.

<summary> O elemento HTML summary (<summary>) é usado como uma ferramenta de onde o usuário irá obter informações adicionais, dentro desse elemento é digitado o 'título', que é o que será escrito na janela, onde o usuário ira clicar para poder observá-las, com isso abrira uma janela onde será exposto o texto adicional com a ajuda do elemento (<p>).

<meta/> - Essa tag permite inserir metadados ao seu documento, no caso acima, a informação charset=”UTF-8″, que garante a compatibilidade do código com os caracteres de padrão latino americano.

<main> – Essas tags representam o conteúdo principal do seu corpo, ou seja, o conteúdo relacionado diretamente com o tópico central da página ou com a funcionalidade central da aplicação.

<aside> – Essas tags representam uma seção de uma página cujo conteúdo é tangencialmente relacionado ao conteúdo do seu entorno, que poderia ser considerado separado do conteúdo;

<nav> – Essa tag define um conteúdo de navegação. Portanto, é muito utilizado em conjunto com listas e na criação de menus

<div> A tag HTML <div> é utilizada para criar blocos dentro da página, para reunir informações e fazer com que o site seja mais fácil de editar dentro do CSS

<span> A tag HTML <span> é utilizada para um propósito parecido à <div>, uma sendo para organizar informações em bloco e outra em linhas, servindo para deixar mais bonito partes específicas da página

<form> A tag HTML <form> é utilizada para criar formulários, assim podendo reunir informações que sejam de interesse ao dono do site.
