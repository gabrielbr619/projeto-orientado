Descobrindo Tags

<select> O elemento HTML select (<select>) apresenta um menu de opções. É um elemento auxiliado por outro 'option'(<option>) e cada um desses 'option' guarda uma opção para o usuário escolher na página. As opções podem também, ser pré-selecionadas para o usuário como 'escolha' assim como se vê nas paginas que vemos no dia-dia.

<figure> O Elemento HTML <figure> representa o conteúdo independente, freqüentemente com o elemento (figcaption), o qual é usado exclusivamente para legendas. Ele é usado para peças que esta isolado corpo principal da página, é independente do fluxo principal.Normalmente, isso é uma imagem, uma ilustração, um diagrama, um trecho de código, mas que pode ser movido para outra página sem afetar o entendimento da página.

<output> O elemento de saída (<output>) é um elemento no qual um site ou aplicativo pode colocar os resultados de um cálculo, através do elemento input, que deve, ou devem, ser preenchidos para a resposta aparecer automaticamente.

<summary> O elemento HTML summary (<summary>) é usado como uma ferramenta de onde o usuário irá obter informações adicionais, dentro desse elemento é digitado o 'título', que é o que será escrito na janela, onde o usuário ira clicar para poder observá-las, com isso abrira uma janela onde será exposto o texto adicional com a ajuda do elemento (<p>).
